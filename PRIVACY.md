# Privacy Policy for Esuyo Markdown

**Last updated:** June 19, 2026

Esuyo Markdown is a desktop application that lets you browse and read markdown files on your computer. This privacy policy explains how the application handles your data.

## Data Collection

Esuyo Markdown **does not collect, store, or transmit any personal data**. The application operates entirely locally on your device.

### What the app accesses:

- **Markdown files (.md, .markdown)** — only in folders you explicitly select via the built-in folder picker dialog. The app reads these files to display them as rendered content.
- **App settings** — your preferences (theme, fonts, recent folders) are stored locally in a `settings.json` file in your system's app data directory. This data never leaves your device.

### What the app does NOT do:

- ❌ No user accounts or registration
- ❌ No telemetry, analytics, or crash reporting
- ❌ No internet communication (except loading fonts — see below)
- ❌ No data sent to external servers
- ❌ No advertisements or tracking
- ❌ No cloud sync or backup
- ❌ No file content sent anywhere

## Network Access

The only network request Esuyo Markdown makes is to **Google Fonts** (`fonts.googleapis.com`, `fonts.gstatic.com`) when the application starts, in order to load the font families you've selected. You can disable this by choosing system fonts instead of web fonts in the settings.

## File Deletion

When you use the "Delete" feature, the file is moved to your operating system's **Recycle Bin / Trash**. It is not permanently erased unless you empty the Trash separately.

## Third-Party Services

This application does not integrate any third-party analytics, advertising, or data processing services.

## Changes to This Policy

If this policy changes, the "Last updated" date at the top will be revised. Since the app does not connect to the internet, you can always find the latest version in the application's source repository.

## Contact

For questions about this privacy policy, please open an issue on the project's GitHub repository.
